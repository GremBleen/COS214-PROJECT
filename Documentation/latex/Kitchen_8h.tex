\doxysubsection{Detailed Description}
Contains declaration for the \mbox{\hyperlink{classKitchen}{Kitchen}} class. 

The \mbox{\hyperlink{classKitchen}{Kitchen}} class is responsible for receiving orders from the \mbox{\hyperlink{classRestaurant}{Restaurant}} class and passing them to the \mbox{\hyperlink{classChef}{Chef}} class. \mbox{\hyperlink{classKitchen}{Kitchen}} holds a vector of \mbox{\hyperlink{classOrderContainer}{Order\+Container}} objects, which are used to pass orders to the \mbox{\hyperlink{classChef}{Chef}} class. \mbox{\hyperlink{classKitchen}{Kitchen}} has a pointer to a \mbox{\hyperlink{classChef}{Chef}} object, which is the chain of responsibility object.
