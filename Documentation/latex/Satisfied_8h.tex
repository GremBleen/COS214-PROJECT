\doxysubsection{Detailed Description}
Contains declaration for the \mbox{\hyperlink{classSatisfied}{Satisfied}} class. 

This class is a concrete implementation of the \mbox{\hyperlink{classRating}{Rating}} class. It is used to represent a satisfied customer. It is used to calculate the tip for a customer. Satified customers tip 25\% of their bill.
