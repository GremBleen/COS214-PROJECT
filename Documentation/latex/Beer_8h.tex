\doxysubsection{Detailed Description}
Contains the declaration for the \mbox{\hyperlink{classBeer}{Beer}} class. 

This file defines the \mbox{\hyperlink{classBeer}{Beer}} class, which is a subclass of the \mbox{\hyperlink{classDrink}{Drink}} class. It contains boolean variables to keep track of whether a beer glass has been obtained, whether beer has been poured into the glass, and whether the beer has been assembled.
