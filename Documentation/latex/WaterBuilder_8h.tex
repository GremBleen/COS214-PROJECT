\doxysubsection{Detailed Description}
Contains declaration for the \mbox{\hyperlink{classWaterBuilder}{Water\+Builder}} class. 

This file contains the declarations for the functions to build a \mbox{\hyperlink{classWater}{Water}} object using the Builder design pattern. \mbox{\hyperlink{classWaterBuilder}{Water\+Builder}} is a concrete builder class that inherits from the \mbox{\hyperlink{classDrinkBuilder}{Drink\+Builder}} class. It is responsible for building a \mbox{\hyperlink{classWater}{Water}} object using the template method pattern.
